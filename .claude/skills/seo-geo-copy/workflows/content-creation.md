## Phase 1: Research (30% of effort)

### Step 1.1: Keyword Research

**Actions:**
1. Identify seed keyword from topic/business need
2. Expand with keyword research tools
3. Analyze metrics (volume, difficulty, CPC)
4. Map keywords to content pieces

**Deliverable:** Keyword list with:
- Primary keyword
- Secondary keywords (3-5)
- Long-tail variations (5-10)
- Question-based queries

### Step 1.2: Search Intent Analysis

**Actions:**
1. Search primary keyword on Google
2. Analyze top 10 results
3. Classify dominant intent:
   - Informational (how-to, guides)
   - Commercial (reviews, comparisons)
   - Transactional (product pages)
   - Navigational (brand searches)
4. Note content format patterns (listicles, guides, etc.)

**Deliverable:** Intent brief:
- Dominant intent type
- Expected content format
- Average word count of competitors
- Common sections/topics covered

### Step 1.3: AI Platform Analysis

**Actions:**
1. Query ChatGPT, Claude, Perplexity with topic
2. Note what sources they cite
3. Identify gaps in AI responses
4. Find questions AI answers poorly

**Deliverable:** AI gap analysis:
- Currently cited sources
- Missing information
- Opportunity areas
- Unique angle possibilities

### Step 1.4: Competitor Content Analysis

**Actions:**
1. Identify top 5 competing content pieces
2. Analyze structure and depth
3. Note unique elements (data, tools, examples)
4. Find content gaps

**Deliverable:** Competitor matrix showing:
- URL, word count, backlinks
- Topics covered
- Unique elements
- Weaknesses/gaps

## Phase 2: Plan (20% of effort)

### Step 2.1: Define Content Specifications

**Document:**
- Content type (article, guide, landing page)
- Target word count
- Primary and secondary keywords
- Target audience/persona
- Unique value proposition
- Call-to-action goal

### Step 2.2: Create Outline

**Structure:**
```
H1: [Title with primary keyword]

Introduction
- Hook
- Answer-first summary
- What reader will learn

H2: [Section 1 - Definition/Background]
H3: [Subsection 1.1]
H3: [Subsection 1.2]

H2: [Section 2 - Core Content]
H3: [Subsection 2.1]
H3: [Subsection 2.2]

H2: [Section 3 - Practical Application]
H3: [Subsection 3.1]
H3: [Subsection 3.2]

H2: FAQ (GEO-optimized)
H3: Question 1?
H3: Question 2?
H3: Question 3?

H2: Conclusion
- Summary
- Key takeaways
- CTA

```

### Step 2.3: Plan GEO Elements

**Identify for each section:**
- Opening statement (answer-first)
- Key entities to define
- Statistics/data to include
- Citable statements to craft

### Step 2.4: Plan Copywriting Elements

**Identify:**
- Headline approach (which formula)
- Introduction hook type
- Social proof to include
- Objections to address
- CTA strategy

---

## Phase 3: Write (30% of effort)

### Step 3.1: Write First Draft

**Approach:**
1. Follow outline structure
2. Write in answer-first format
3. Focus on substance over style
4. Include all planned elements
5. Don't self-edit while drafting

**Per Section:**
```
1. State the answer/key point first
2. Elaborate with details
3. Support with evidence/examples
4. Define any new entities
5. Connect to next section
```

### Step 3.2: Enhance with Copywriting

**Apply:**
1. Hook in introduction
2. Benefit-focused language
3. "You" and "your" for direct address
4. Power words where appropriate
5. Transitions for flow

### Step 3.3: Add Citation-Worthy Elements

**Ensure each major section has:**
- At least one specific statistic
- A clear, quotable statement
- Proper source attribution
- Concrete examples

### Step 3.4: Craft Meta Elements

**Write:**
- Title tag (≤60 chars, keyword-first)
- Meta description (≤155 chars, CTA)
- H1 (can differ slightly from title tag)
- Image alt text

---

## Phase 4: Optimize (15% of effort)

### Step 4.1: SEO Pass

**Check and adjust:**
- [ ] Primary keyword in title, H1, intro
- [ ] Secondary keywords distributed
- [ ] Header hierarchy correct
- [ ] Internal links added (3-5)
- [ ] External links added (2-3)
- [ ] Image alt text complete
- [ ] URL structure clean

### Step 4.2: GEO Pass

**Check and adjust:**
- [ ] Answer-first format everywhere
- [ ] Self-contained chunks
- [ ] Entity clarity (no ambiguity)
- [ ] Statistics sourced
- [ ] FAQ section present

### Step 4.3: Copywriting Polish

**Check and adjust:**
- [ ] Headline compelling
- [ ] Opening hooks reader
- [ ] Benefits clear
- [ ] CTA prominent and clear
- [ ] Flow and readability

### Step 4.4: Add Schema Markup

**Implement:**
- Article/BlogPosting schema
- FAQ schema (for FAQ section)
- Author schema
- Organization schema

---

## Phase 5: Validate (5% of effort)

### Run Checklists

**Complete:**
- [ ] Pre-publish checklist (all passes)
- [ ] Schema validation (Google tool)
- [ ] Mobile preview check
- [ ] Link testing

### Peer Review (Optional)

**Review for:**
- Factual accuracy
- Readability
- Persuasiveness
- Technical accuracy

---

## Phase 6: Publish

### Publication Steps

1. **Final preview** on staging/preview
2. **Publish** content
3. **Submit to Google** via Search Console
4. **Submit to Bing** via Webmaster Tools
5. **Trigger IndexNow** (if implemented)
6. **Update sitemap** (usually automatic)

### Promotion (Optional)

- Share on social media
- Send to email list
- Internal team notification
- Outreach for links

---

## Phase 7: Monitor

### Week 1-2

- Check indexing status (Google/Bing)
- Monitor initial rankings
- Check for crawl errors
- Test AI platform citations

### Month 1

- Track keyword rankings
- Monitor traffic trends
- Analyze engagement metrics
- Note AI citation frequency

### Ongoing

- Quarterly content audits
- Refresh outdated content
- Update based on performance data
- Iterate and improve

## Workflow Variations

### Quick Content (1,000-1,500 words)

Streamlined process:
1. Light keyword research (30 min)
2. Brief outline (15 min)
3. Write + optimize in one pass (2 hours)
4. Quick validation (15 min)

### Pillar Content (3,000+ words)

Extended process:
1. Deep research (1-2 days)
2. Detailed outline with team review (half day)
3. Write in sections over multiple sessions (2-3 days)
4. Multiple optimization passes (half day)
5. Thorough validation + peer review (half day)

### Update/Refresh Existing Content

Targeted process:
1. Audit existing content (1 hour)
2. Identify gaps and outdated sections (30 min)
3. Update specific sections (1-2 hours)
4. Re-optimize for current SEO/GEO standards (1 hour)
5. Revalidate and republish (30 min)

## Common Pitfalls

### Research Phase
- Skipping competitor analysis
- Ignoring AI platform analysis
- Choosing wrong intent match

### Planning Phase
- No clear unique value
- Vague outline without details
- Missing GEO considerations

### Writing Phase
- Context-dependent content
- Missing statistics/data
- Weak introduction hook

### Optimization Phase
- Over-optimizing keywords
- Forgetting schema markup
- Weak CTAs

### Publication Phase
- Not submitting to Bing
- No promotion plan
- Not tracking from day 1
